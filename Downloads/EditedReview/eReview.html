Skeletal Tags Definition
html The root of an HTML document.
head The information about the document.
meta The metadata about an HTML document.
body The document's body.
Text Tags (block-level) Definition
p A paragraph.
h1 to h6 HTML headings.
blockquote A section that is quoted from another source.
address Contact information for the author/owner of a document.
pre Preformatted text.
Text Tags (Inline) Definition
em Emphasized text.
strong Important text.
q A short quotation.
cite The title of a work.
dfn The defining instance of a term.
code A piece of computer code.
mark Marked/highlighted text.
del Text that has been deleted from a document.
ins A text that has been inserted into a document.
abbr An abbreviation or an acronym.
span A section in a document.
br A single line break.
Lists (block-level) Definition
ol An ordered list.
ul An ordered list.
li A list item.
dl A description list.
dt A term/name in a description list.
dd A description/value of a term in a description list.
Structural Elements Definition
header A header for a document or section.
footer A footer for a document or section.
nav Navigation links.
Section A section in a document.
article An article.
main The main content of a document.
aside Content aside from the page content.
div A section in a document.
hr A thematic change in the content.
Links and Media Tags Definition
a A hyperlink.
link HTML headings.
img A section that is quoted from another source.
figure It specifies self-contained content.
figcaption A caption for a figure element.
video A video or movie.
audio It is sound content.
object An embedded object.
param A parameter for an object.
embed A container for an external (non-HTML) application.
Tables Definition
table A table.
tr A row in a table.
th A header cell in a table.
td A cell in a table.
caption A table caption.
thead Groups the header content in a table.
tbody Groups the body content in a table.
colgroup It specifies a group of one or more columns in a table for formatting.
col It specifies column properties for each column within a element.
tfoot Groups the footer content in a table.
More Tags Definition
Style Style information for a document.
Script a client-side script.
Canvas Used to draw graphics, on the fly, via scripting (usually JavaScript).
svg A container for SVG graphics.
